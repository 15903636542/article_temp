## 内置功能：

1：不同角色提供不同服务（可以自定义）
2：发布、审核、管理、导出论文
3：添加、修改、查看通知或公告



## 角色功能：

管理员：增删改查（用户、论文、公告、角色、时间管理）
教师：增删改查（论文、公告、时间管理）
学生：查（论文、公告、时间管理）、选论文、提交论文材料
